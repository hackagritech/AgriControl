# AgriControl #

#### Requisitos ####

* PHP 7 ou posterior
* Version 0.0.1

#### Recursos utilizados ####
 * PHP
 * JavaScript
 * Firebase
 * Git 
 * CSS
 * HTML

#### Executar Aplicação ####

Navegue até a pasta raiz do projeto,
em seguida abra o terminal e digite:

`` php -S localhost:80  ``

Abra o browser e digite:  http://localhost:80
